# mizar-frontend: Source Loading

> 正本は英語です。英語版: [../en/source.md](../en/source.md)。

状態: planned。

## 目的

このモジュールは、前処理と字句解析より前の `.miz` ファイルのソース読み込みを定義する。

## 責務

- ディスクまたはエディタ提供バッファからソースバイト列を読む。
- UTF-8 を検証し、許可される場合は先頭の UTF-8 BOM を 1 つ取り除く。
- パッケージ相対のモジュールパスを導出する。
- ソースハッシュを計算する。
- `mizar-session` のソース同一性型を使って line map と loading map を構築する。
- 構文解析を行わず、ファイルレベルの診断を生成する。
