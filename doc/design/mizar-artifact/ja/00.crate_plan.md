# Crate Plan: mizar-artifact

> 正本は英語です。英語版:
> [../en/00.crate_plan.md](../en/00.crate_plan.md)。

## 責務

所有する挙動:

- 下流パッケージ、LSP、ドキュメント、AI tooling 向けの安定した公開 artifact
  射影。
- `ModuleSummary`、`RegistrationSummary`、`ProofWitnessRef`、
  `VerifiedArtifact` などの artifact 向けスキーマ。
- 公開スキーマが共有する正準シリアライゼーションと artifact ハッシュ入力。
- artifact store の layout、atomic write、検証つき reader、manifest
  transaction。
- content hash による proof witness 参照と、公開 artifact の再現・調査に必要な
  provenance field。

範囲外:

- raw `SurfaceAst`、typed AST、core IR、control-flow IR、VC IR、ATP problem、
  kernel state の dump。
- internal cache record、cache key の所有、cluster-db storage、proof reuse
  validation、scheduler state。
- proof authority、proof search、kernel acceptance、policy decision。
- resolver、checker、VC、kernel、proof crate 内の producer-owned projection
  logic。

## Exit Criteria

Hard gates:

- blocking/high の specification inconsistency が残っていない。
- source behavior が `doc/spec`、`.miz` tests、traceability metadata、または
  task-specific module spec の外に残っていない。
- 文書化された artifact/source responsibility boundary が守られている。
- raw IR、cache record、scheduler state、proof authority、kernel acceptance behavior を
  この crate が所有していない。
- 完了した各 task が独立した commit を持つ。
- 残る gap は `deferred`、`external_dependency_gap`、または human-owned に分類されている。

Verification commands:

- Rust source task ごと:
  - `cargo test -p mizar-artifact`
  - `cargo clippy -p mizar-artifact --all-targets -- -D warnings`
  - `cargo fmt --check`
- task 1 の新しい workspace member:
  - `cargo check --workspace`
- crate 完了の広範囲確認:
  - `cargo clippy --all-targets --all-features -- -D warnings`
  - `cargo test`

Review expectations:

- implementation specification / documentation の review-only agent。
- test sufficiency の review-only agent。
- full implementation の review-only agent。
- source/documentation consistency の review-only agent。
- hard-gate status と 90/100 以上の quality score を持つ最終 crate completion review。
