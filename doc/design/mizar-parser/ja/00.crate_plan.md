> canonical languageはEnglish。English canonical:

## Responsibility

`mizar-parser`はfrontend tokenを消費し、parser-owned syntax event、recovery
diagnostic、`mizar-syntax` surface treeを生成する。concrete grammar recognition、
source-order preservation、grammar boundary recoveryを所有する。

name resolution、type checking、proof acceptance、semantic evidence、CoreIr、
ControlFlowIr、VC generation、既存parser vocabulary外のpublic diagnostic code
allocationは所有しない。

## Task Decomposition

### Task 47: `reconsider` justification parser alignment

purposeはP-265-47A/B/Cだけをcloseすること。

authority sliceはEnglish Chapters 4 §4.4.2、8 §8.2.2、15 §§15.5.1/15.8.2/
15.12、Appendix A §§A.4/A.15、exact deferred trace row、既存`.miz` control。

consumer/visibilityは既存private `parse_reconsider_statement_at` producerと既存
parse-only runner。public syntax node kind/accessorは変更しない。

required behavior:

- `reconsider ... as T;`はjustification childもparser diagnosticもない
  `ReconsiderStatement`を生成する。
- `reconsider ... as T by refs;`は`JustificationClause`を1件維持する。
- `reconsider ... as T proof ... end;`は既存missing-`end` recoveryを含む
  `ProofBlock`を1件維持する。`ProofBlock`は`proof`、reasoning body、`end`を
  ownershipし、`ReconsiderStatement`はfinal semicolonをownershipする。
- bare/equated item、target type ownership、source order/range、`then` wrapping、
  explicit-simple behavior、malformed item/type recoveryは不変。
- `consider`はmandatory-simple-justification hostのまま。

tests:

- planned test-first pass fixture/sidecarを追加する。
- 3 tail form、AST child ownership、recoveryなしのomission、proof-block
  missing-`end` recovery、unchanged `consider`/malformed item behaviorをunit testする。
- proof blockをrejectするhostの既存unit-test listから`reconsider`だけを外し、真に
  simple-onlyな全hostはguardし続ける。
- existing mixed recovery expectation/unit countはobsolete omitted-tail errorだけを更新。
- Task-47 trace row exactly 2件だけをactivateし、general Chapter-15 statement rowへ
  new pass caseを追加する。

source edit前にaffected design pathを次へ固定する:

- `doc/design/mizar-parser/README.md`;
- parser paired `00.crate_plan.md`、`README.md`、`grammar.md`、`recovery.md`、
  `source_spec_audit.md`、`todo.md`、
  `bilingual_documentation_synchronization.md`;
- mizar-test paired `00.crate_plan.md`、`todo.md`、`harness.md`、
  `traceability.md`、`bilingual_sync_audit.md`、`module_boundary_audit.md`;
- `doc/design/todo.md`と`doc/design/spec_coverage_audit.md`。

parser `pratt.md`、他component design、historical current-state snapshotはout of scope。
source/test pathは正確に`crates/mizar-parser/src/module.rs`、
`crates/mizar-parser/src/module/tests.rs`、`crates/mizar-test/tests/metadata.rs`、
