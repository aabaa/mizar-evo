# Bialystok Presentation, September 2026

This directory is for a planned presentation to the Mizar team in Bialystok.

The initial artifact is a Markdown content draft. Beamer source should be added
after the talk structure, examples, and collaboration questions stabilize.

## Planned Artifacts

- `draft.md` - second English content draft with a more detailed argument,
  code comparison plan, architecture story, migration roadmap, and Formalized
  Mathematics model.
- `draft.ja.md` - Japanese companion draft for faster review.
- `main.tex` - future Beamer source.
- `references.bib` - future bibliography.
- `figures/` - diagrams, screenshots, architecture figures, and generated
  visual assets.

## Working Assumptions

- The target audience knows current Mizar well.
- The talk should be in English.
- Speaker notes can later include Japanese planning notes if useful.
- Code comparisons should be verified against real legacy Mizar examples before
  the final Beamer version.
