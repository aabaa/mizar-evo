\begin{frame}[fragile,t]{1.1 - Three Pillars (1/2)}
The design is judged by three constraints that must hold at the same time.

\begin{frame}[fragile,t]{1.3 - First-Order Core As A Strategic Choice}
\smallskip
\textbf{Key Points:}
\begin{itemize}
\setlength{\itemsep}{1pt}
\setlength{\parsep}{0pt}
\setlength{\parskip}{0pt}
\item A first-order set-theoretic core matches much ordinary mathematical practice.
\item ATP tools are strong in first-order reasoning.
\item Soft types, modes, attributes, and registrations can remain Mizar-facing structure around that core.
\item The kernel can focus on checking evidence rather than performing high-level elaboration.
\end{itemize}
\smallskip
\textbf{Design Reason:}
\begin{itemize}
\setlength{\itemsep}{1pt}
\setlength{\parsep}{0pt}
\setlength{\parskip}{0pt}
\item This choice preserves the current Mizar foundation while letting powerful ATPs work at the layer where they are strongest.
\item It also avoids making dependent-type elaboration or tactic execution part of the trusted explanation of ordinary mathematical text.
\end{itemize}
